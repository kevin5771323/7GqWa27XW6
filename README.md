# 人力资源管理系统 java370

## 项目概述

人力资源管理系统是一个基于 JavaWeb 技术的简单项目，适用于对部门信息、员工信息、薪资信息进行管理。项目采用前后端不分离的架构，主要包括基础的增删改查操作，以及员工信息导出为 Excel 文件的功能。

## 技术栈

- Java
- JSP
- MySQL
- Tomcat

## 功能模块

- 管理员登录
- 部门信息管理
  - 增删改查操作
- 员工信息管理
  - 增删改查操作
  - 导出为 Excel 文件
- 薪资信息管理
  - 增删改查操作

## 系统角色

- 管理员：拥有所有操作权限

## 运行环境

- 开发工具：IntelliJ IDEA
- 服务器：Tomcat
- 数据库：MySQL

## 目录结构

```
project/
├── WebContent/
│   ├── WEB-INF/
│   │   ├── classes/
│   │   ├── lib/
│   │   └── web.xml
│   └── ...
└── src/
    └── ...
```

## 核心亮点

- 简单易用的用户界面
- 支持员工信息导出为 Excel 文件
- 代码结构清晰，易于维护和扩展

## 部署步骤

1. 安装并配置 MySQL 数据库
2. 部署 Tomcat 服务器
3. 部署项目到 Tomcat 的 WebApps 目录
4. 启动 Tomcat 服务器，访问项目

注意：具体部署步骤请参考详细部署文档。在部署过程中应确保遵守相应操作规范，保障系统安全。
